Rodeo is a data centric IDE for Python. You can think of it as an alternative 
UI to the notebook for the IPython Kernel. It's heavily inspired by great projects 
like RStudio and Eclipse.

## Quickstart
```bash
$ pip install rodeo 
$ rodeo .
```
